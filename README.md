# 経営ダッシュボード（フリーランスWebコーダー専用）

`requirements.md` の要件定義に基づく、個人利用の経営ダッシュボードです。
売上・利益・案件・固定費・自己投資・入金・税金積立・キャッシュフローを1つのアプリで管理します。

## 起動方法

**`index.html` をブラウザで開くだけ**です。ビルド・サーバー・インストールは不要です。

データはブラウザの localStorage に保存されます（このブラウザ内のみ・単一ユーザー前提）。
初回は「設定 → データ管理 → サンプルデータを投入」で動作を確認できます。

## 画面構成

| タブ | 内容 | 対応要件 |
|---|---|---|
| ダッシュボード | カードの表示/非表示・ドラッグ＆ドロップ並び替え（保存される）、入金遅延の警告表示 | FR-401〜403, FR-105/106 |
| 入金管理 | 請求のCRUD・ステータス絞り込み・「入金済みにする」ワンクリック・案件選択で金額自動補完 | FR-101〜106 |
| 案件 | 案件管理（契約金額・案件経費・利益率） | 既存機能 |
| 固定費・サブスク | 月額費用の管理 | 既存機能 |
| 自己投資 | 書籍・講座・機材などの記録 | 既存機能 |
| キャッシュフロー | 残高・今月収支・予測残高、月次収支/残高推移グラフ | FR-301/302 |
| 税金積立 | 推定納税額（所得税・住民税・個人事業税・国保・消費税）と月次積立推奨額 | FR-201〜204 |
| 設定 | 売上目標、税率・控除・速算表の変更（即時反映）、エクスポート/インポート | FR-205 ほか |

## 設計メモ

- 依存ライブラリなし（Vanilla JS）。`file://` で直接開けるよう ES modules ではなく通常の script 読み込み。
- **DR-01**: 金額はすべて円単位の整数で保持し、表示時のみ3桁区切り。
- **DR-02**: 未入金総額・遅延・推定納税額・予測残高などの集計値は保存せず `js/calc.js` で都度算出。
- **NFR-02**: ダッシュボードのカードは `js/cards.js` のレジストリに1件追加するだけで増やせる
  （`store.js` の `DEFAULT_CARD_ORDER` にキーを追加すると表示候補に載る）。
- 売上は**請求ベース**（請求日・請求金額）、キャッシュフローは**入金ベース**（入金日・入金金額）で区別して集計（DR-04）。
- 税額試算は概算・参考値です。正確な税額は確定申告で確定します（FR-204）。

## ファイル構成

```
index.html        エントリポイント
css/style.css     スタイル（ライト/ダーク対応）
js/utils.js       整形・日付ユーティリティ
js/store.js       localStorage 永続化・既定値・サンプルデータ
js/calc.js        集計ロジック（都度算出）
js/tax.js         税金試算
js/cards.js       ダッシュボードカードのレジストリ
js/views.js       各タブの画面・フォーム・グラフ
js/app.js         タブ切り替え・初期化
```
